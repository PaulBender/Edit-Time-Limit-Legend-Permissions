## INFORMATION ##

Edit Time Limit/Legend Permissions (1.0.2) for MyBB 1.6
Created by: Starpaul20
Copyright: ©2011
License: GPL

Adds two usergroup permissions that allow you to:
-Limit the length of time that users have to edit their own posts.
-Allowing the disabling of the edited by legend whenever you edit (please note that you may only disable the legend in full edit mode, it cannot be disabled in quick edit).

Please note that when you use this plugin, please make sure the setting 'Edit Time Limit' is set to 0 (to prevent any collisions).

The usergroup permissions are located under the Editing/Deleting Options section of the 'Forums and Posts' tab when editing a usergroup. This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Install & Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you only need to overwrite old files with the new ones, no need to deactivate.
